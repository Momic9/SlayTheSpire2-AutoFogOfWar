# 🌫️ oneClickFogWar

一款基于图像识别的杀戮尖塔2屏幕自动涂抹工具，专为需要"一键释放战争迷雾"场景设计。程序持续截取屏幕，一旦检测到预设图标（怪物、精英、商店、火堆、问号等），便自动用鼠标右键在该位置涂抹。
Made by anthropic/claude-sonnet-4.6

---

## ✨ 功能特性

- 🔍 **实时屏幕检测**：每隔固定间隔截取全屏，使用 OpenCV 模板匹配定位目标图标
- 🖱️ **自动右键涂抹**：以检测到的图标为中心，按住右键进行 S 形扫描涂抹
- 🧩 **多模板支持**：自动加载 `source/` 目录下所有 PNG 图片作为检测模板
- ⏸️ **随时暂停/继续**：按 `F9` 键切换暂停与运行状态
- ❌ **快速退出**：按 `ESC` 键或将鼠标移至屏幕左上角立即停止
- 🕒 **冷却机制**：同一位置涂抹后进入冷却，避免重复操作

## 🛠️ 环境要求

- Python 3.10+
- Windows 操作系统（依赖 `ctypes.windll` 和 `ImageGrab`）

### 安装依赖

```bash
pip install opencv-python numpy pyautogui Pillow
```

---

## 🚀 使用方法

1. 将需要检测的图标截图保存为 PNG 格式，放入 `source/` 目录
2. 运行主程序：

```bash
python fog_war.py
```

3. 切换到目标游戏/应用窗口，程序将自动开始检测并涂抹

### 快捷键

| 按键 | 功能 |
|------|------|
| `F9` | 暂停 / 继续 |
| `ESC` | 退出程序 |
| 鼠标移至左上角 | 紧急停止（PyAutoGUI 安全机制） |

---

## ⚙️ 参数配置

在 `fog_war.py` 顶部的配置区域可调整以下参数：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `SCAN_INTERVAL` | `0.5` 秒 | 截图检测间隔 |
| `MATCH_THRESHOLD` | `0.80` | 模板匹配置信度阈值（0~1，越高越严格） |
| `PAINT_SIZE` | `150` px | 涂抹区块边长 |
| `PAINT_DURATION` | `0.1` 秒 | 每次涂抹持续时间 |
| `PAINT_STEP` | `3` px | 涂抹扫描步长 |
| `COOLDOWN` | `2.0` 秒 | 同一位置涂抹后的冷却时间 |

---

## 📝 注意事项

- 模板图片应尽量清晰，与游戏中实际显示的图标大小一致，以提高匹配准确率
- 若误触发率较高，可适当提高 `MATCH_THRESHOLD`（如调至 `0.85` 或更高）
- 程序运行期间请勿遮挡目标窗口
- 本工具仅供学习与个人娱乐使用
